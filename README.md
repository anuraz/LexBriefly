# Lex Briefly

Lex Briefly is a React + TypeScript law study platform for Delhi University LL.B students, featuring academic resources, case summaries, internships, and a student login system.

## Features
- Student email-based login
- Academic portal with study materials
- Landmark cases database
- Internship and legal news sections
- Admin CMS interface

## Tech Stack
- React
- TypeScript
- Vite
- Express
- Tailwind CSS

## Development

Install dependencies:
```bash
npm install
```

Run locally:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
